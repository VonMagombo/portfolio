## Getting started

Prerequisites

- Node.js (if the project uses a build tool)
- npm or yarn

Install

```bash
# Clone the repo
git clone https://github.com/VonMagombo/portfolio.git
cd portfolio

# Install dependencies (if applicable)
npm install
# or
# yarn install
```

Run (development)

```bash
npm start
# or
# npm run dev
```

Build

```bash
npm run build
```

Deploy

- Deploy the contents of the build/ or dist/ folder to Netlify, Vercel, or GitHub Pages.
